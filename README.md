# Only-Bones-for-Twitter
在新标签页优美排版阅读X推文。Turn any X/Twitter thread into a clean, immersive, and distraction-free blog post with one click.一键剔除冗余，将推文转化为纯净、沉浸式的 Blog 阅读模式

![Only-Bones Icon](icon.png)

## 💡 核心理念 (Why Only-Bones?)
推特（X）杂乱的界面很烦！充斥屏幕的广告、侧边栏、推荐趋势、无用的弹窗以及复杂的指标（浏览量/点赞数），正在严重干扰我们的阅读体验。

不知道你是不是和我一样喜欢特别优美地复制粘贴保存图文，或者其优美地排版截图。目前市面上没有这么一款插件或者软件，所以自己就开发了一个。

Tired of the cluttered X (Twitter) interface! Ads, sidebars, pop-ups, and vanity metrics severely disrupt the reading experience. Unable to find a tool that allows for **"elegant copy-pasting and perfect screenshots,"** I built Only-Bones.

Using a "dimensional reduction" approach, it acts like a scalpel—extracting only the core "bones" of a tweet and reconstructing them into an ultra-clean, centered, standalone blog page in a new tab.

**Only-Bones** 采用“降维提取”的核心理念。它不只是简单地隐藏网页元素，而是像手术刀一样，精准提取推文最核心的内容（骨架），并在新标签页为你重构一个极其干净、居中的独立 Blog 页面。

## 🎯 谁需要 Only-Bones？ (Who is this for?)
* 📖 **深度阅读者 (The Deep Reader)**
    厌倦了信息流的喧嚣，渴望在留白中沉浸式阅读长推文（Thread）。
    *Craves an immersive, distraction-free environment for reading long threads.*
* 🗂️ **数字档案员 (The Digital Archivist)**
    知识整理强迫症，需要将高价值图文无痕、优雅地复制归档到 Notion 或 Obsidian。
    *Needs to flawlessly copy and save high-value content into personal knowledge bases like Notion or Obsidian.*
* 🎨 **排版美学控 (The Aesthetic Sharer)**
    极简主义信徒，只想要一张宛如艺术品般纯净、无侧边栏的推文截图用于分享。
    *A minimalist who wants perfectly formatted, clutter-free screenshots of tweets for sharing.*
* 🔍 **专注探寻者 (The Focused Seeker)**
    拒绝被“转评赞”等虚荣指标裹挟，直取信息核心与信源。
    *Ignores vanity metrics (likes/retweets) to focus purely on the core information.*

## ✨ 核心功能 (Features)
* **纯净提取：** 智能抓取作者头像、ID、发布时间、文本和高清图片，彻底抛弃原有框架。
* **沉浸式阅读：** 采用最符合人眼视距的黄金宽度和居中排版，找回深度阅读的平静感。
* **评论区收纳：** 完美保留原贴的评论盖楼逻辑，自带“一键折叠/展开”的黑线开关，只看你想看的。
* **绝对安全：** 0 追踪，0 广告，0 数据上传。所有提取动作仅在你的浏览器本地进行。

Smart Extraction: Accurately captures the author's avatar, handle, timestamp, text, and high-res images, leaving all the site infrastructure behind.
Immersive Reading: Uses an optimized, centered layout with the perfect reading width to help you focus deeply.
Comment Control: Perfectly preserves the thread's comment hierarchy, featuring a sleek, one-click toggle to collapse or expand the reply section.
100% Privacy: Zero tracking, zero ads, and zero data collected. All extraction happens entirely locally within your browser.

## 🛠️ 如何安装 (How to Install)
由于本项目未上架 Chrome 商店，请通过以下开发者模式安装：

1. 点击网页右上角的 **Code** 按钮，选择 **Download ZIP**，下载并解压到你的电脑上。
2. 打开 Chrome / Edge 浏览器，在地址栏输入 `chrome://extensions/`（Edge 为 `edge://extensions/`）并回车。
3. 开启页面右上角的 **开发者模式 (Developer mode)**。
4. 点击左上角的 **加载已解压的扩展程序 (Load unpacked)**。
5. 选择你刚才解压的文件夹，即可安装成功！
6. **建议：** 点击浏览器右上角的拼图图标，将 🦴 鱼骨头图标固定在工具栏，方便随时调用。

## 🚀 使用方法 (Usage)
打开任意一条带有 `/status/` 的推特/X 推文详情页，点击右上角的“鱼骨头”图标，瞬间进入属于你的纯净阅读世界！

Open any specific tweet/thread page on X/Twitter, click the "Fish Bone" icon in your extension bar, and instantly enjoy a pure reading experience!
